# Calculator App

A sleek, responsive web calculator with basic arithmetic, decimal support, and a dark mode toggle. Built with vanilla JavaScript for simplicity and performance.

## Features
- Basic operations (+, -, *, /).
- Decimal numbers.
- Dark/light mode toggle.
- Responsive design.

## How to Use
1. Clone the repo.
2. Open `index.html` in a browser.
3. Click buttons to calculate.

## Deployment
Host on GitHub Pages for a live demo.

## Future Enhancements
- Add scientific functions.
- Keyboard support.
